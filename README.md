# n8n-nodes-linkedinoutreach

This is an n8n community node. It lets you use LinkedIn Outreach by Boost.space in your n8n workflows.

LinkedIn Outreach by Boost.space enables automated LinkedIn messaging and connection requests through the Unipile API, allowing you to scale your LinkedIn outreach campaigns.

[n8n](https://n8n.io/) is a [fair-code licensed](https://docs.n8n.io/reference/license/) workflow automation platform.

## Installation

Follow the [installation guide](https://docs.n8n.io/integrations/community-nodes/installation/) in the n8n community nodes documentation.

## Operations

### User
- **Send LinkedIn Connection Request**: Send a connection request to a LinkedIn user with an optional personalized message.

### Message
- **Send Message**: Send a new message to start a conversation with one or more LinkedIn users. Supports:
  - Classic LinkedIn API
  - Sales Navigator API
  - Recruiter API
  - File attachments (PDF, images)
  - InMail messages

## Credentials

To use this node, you need to obtain credentials from the AI Outreach dashboard:

1. **User ID**: Your User ID shown in the AI Outreach dashboard
2. **API Key**: Your API Key shown in the AI Outreach dashboard
3. **Account ID**: The LinkedIn account ID to use for outreach

## Compatibility

- Minimum n8n version: 1.0.0
- Tested with n8n version 1.98.0
